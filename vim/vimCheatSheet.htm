BASICS

FIND

:wq   

write and quit

/bird

finds a bird!

:q!     

quit without writing

p / n

takes you to the previous / next bird

i    

enter Insert mode from Command mode

f*

to the next '*' in the line (not above or below)

esc   

exit Insert mode & return to Command mode

F*

previous '*'

v

from command mode to visual mode

;

scroll through your f command

u

undo!

REPLACE

ctl + r  

redo

:%s/bird/cat/gc

replace all the birds with cats, but ask first

DIRECTIONS

:2,5s/bird/cat/g

between lines 2 & 5, replace all birds with cats, no asking

h     

to move the cursor one space left

:%s_bird_cat_g

replace all the birds with cats (changing the / to something else helps w/ pathnames)

j    

to go down one line

SCROLL

k    

to go up one line

control + b / f

takes you back / forward a page

l    

to go one space right

control + d / u

takes you down / up half a page

I LIKE TO MOVE IT MOVE IT

GETTING IN GOOD

gg / G

takes you to the beginning / end of the file

I / A

insert beginning / end of current line

0 / $

takes you to the beginning / end of the line

P

pastes *before* your cursor (yy includes a /n)

w / b

takes you to the next / previous word

o / O

adds a new line below / above the current line and goes into insert mode

DELETE!

MOVES LIKE JAGGER

dw

to delete a word

) / (

forward / backward 1 sentence

d$

to delete from where you are to the end of the line

} / {

forward/backward one paragraph

VISUAL EFFECTS

''

move to the last line you jumped from (2 apostrophes)

y  

ends and yanks your selection

DUAL WEILDING

p  

pastes your selection

:e /OMG/WTF

while already in vim, open up /OMG/WTF for editing

yy

yanks the line you are on

:bp / :bn

move to previous / next opened file to edit

yap

yanks around paragraph

:bp / :bn

move to previous / next opened file to edit
